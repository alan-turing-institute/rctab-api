Welcome to RCTab API's documentation!
=====================================

The RCTab API is a FastAPI web server that provides a frontend for users and an API for the CLI & function apps.
It requires a PostgreSQL database, in which it will store subscription data.
It can be run locally, for development, or deployed to Azure with the Infrastructure code.
